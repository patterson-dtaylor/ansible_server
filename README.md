# Ansible Server Build

## Installing
1. Set GitHub Token:
    1. ```bash
        export TOKEN=<Your GitHub token>
        ```
2. Clone repo:
    1. ```bash
        git clone https://${TOKEN}:x-oauth-basic@github.com/patterson-dtaylor/ansible_server.git
        ```
3. Run Install Script:
    1. ```bash
        ./install.sh
        ```
